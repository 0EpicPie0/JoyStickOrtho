# XIAO-ESP32C6 Web Joystick

Скетч: `xiao_esp32c6_web.ino`

## Что делает

- Инициализирует `WiFi AP` (точка доступа).
- Поднимает HTTP-сервер на `:80`.
- Отдаёт web-интерфейс тестирования джойстика.
- Отдаёт текущее состояние джойстика в JSON через `/data`.

## Основные endpoint'ы

- `/` — HTML + JS интерфейс
- `/data` — JSON с сырыми и нормализованными значениями (`xr`, `yr`, `xn`, `yn`, `dir`)

## Примечания

- В скетче есть калибровка, сглаживание и dead zone.
- Логика рассчитана на запуск в режиме точки доступа (без внешнего роутера).
