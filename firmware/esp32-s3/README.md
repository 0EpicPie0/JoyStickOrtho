# ESP32-S3 (утраченная прошивка)

Оригинальная прошивка ESP32-S3 в этом репозитории отсутствует (утеряна).  
Ниже зафиксирован протокол, который ожидает desktop-приложение.

## Serial-протокол

- Скорость: `9600 baud`
- Формат: одна JSON-строка на пакет
- Разделитель пакетов: `\n` или `\r\n`

### Обязательные поля

- `x` (number)
- `y` (number)
- `cmd` (string)

### Пример пакета

```json
{"x":2048,"y":1800,"cmd":"MOVE"}
```

## Как используется

Desktop-приложение (`desktop/joystick-visualizer`) читает строку из serial-порта, парсит поля `x`, `y`, `cmd` и визуализирует координаты.

`cmd` сохраняется при парсинге и может использоваться для расширения логики (режимы рисования, команды управления и т.п.).
